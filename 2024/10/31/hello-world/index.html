在 HTML 文档的第一行通常是文档类型声明，具体来说是 <!DOCTYPE html>。这个声明告诉浏览器使用 HTML5 标准来解析和显示网页内容。它的作用包括：

  1. 确保浏览器使用标准模式：<!DOCTYPE html> 声明会触发浏览器的标准模式，而不是怪异模式（quirks mode）。标准模式下，浏览器会尽可能遵循 W3C 的规范，而怪异模式可能会使用一些过时的、不标准的行为来解析和显示网页。
  2. 提高兼容性：使用 <!DOCTYPE html> 可以确保网页在不同浏览器中的表现更一致，减少由于浏览器差异导致的显示问题。
  3. 优化性能：在标准模式下，浏览器的渲染引擎可以更有效地工作，从而提升页面的加载和渲染速度。

这些标签的合理使用可以提升网页的功能性、可读性和SEO效果，是编写高质量 HTML 文档的重要组成部分。

<meta> 标签用于在 HTML 文档中提供关于网页的元数据。这些元数据不会直接显示在页面上，但它们在很多方面都非常重要，包括搜索引擎优化（SEO）、浏览器行为和网页加载性能。常见的 <meta> 标签包括：

  1. 字符集声明：指定网页使用的字符编码。

  2. 描述：提供关于网页内容的简短描述，通常用于搜索引擎结果页面。

  3. 关键字：列出与网页内容相关的关键字，帮助搜索引擎理解页面主题。

  4. 作者：声明网页的作者。

  5. 视口设置：用于控制网页在移动设备上的显示和缩放行为。

通过合理地使用 <meta> 标签，可以显著改善网页的用户体验、兼容性和可发现性。
